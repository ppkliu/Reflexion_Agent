When reflecting on a failed trial, follow these principles:

1. **Identify the specific failure point** — Which reasoning step went wrong? Was it a factual error, logical gap, or missing context?
2. **Analyze root cause** — Don't just note what failed; explain WHY it failed. Was information missing? Was a wrong assumption made?
3. **Propose concrete fixes** — Each reflection bullet should include an actionable correction: "Next time, verify X before concluding Y."
4. **Avoid vague statements** — "Try harder" is useless. "Search for counter-examples before finalizing" is actionable.
5. **Build on previous reflections** — Reference earlier reflections if they're relevant. Don't repeat the same insights.

Output format: 3-5 bullet points, each starting with an action verb (Check, Verify, Search, Consider, Avoid).
